== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Live Vote'
3. Activate Live Vote from your Plugins page. (You will be greeted with a Welcome page.)

= From WordPress.org =

1. Download Live Vote.
2. Upload the 'Live Vote' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Live Vote from your Plugins page. (You will be greeted with a Welcome page.)
